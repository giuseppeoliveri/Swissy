Swissy è uno script per Adobe InDesign che automatizza l'alternanza ritmica tra righe lunghe e corte. È sufficiente impostare la larghezza desiderata per le righe pari e dispari, lasciando che l'algoritmo ottico calcoli le interruzioni migliori per ottenere una texture tipografica perfetta e ordinata.

Come si installa: Inserisci il file .jsx direttamente nella cartella Scripts Panel di Adobe InDesign.

Inserisci anche il png se vuoi la mascotte!

(Solitamente il percorso è: Applicazioni > Adobe InDesign > Scripts > Community > Scripts Panel)

Seleziona una casella di testo con il testo in giustificato e divertiti con Swissy!

//ENG:

Swissy is a script for Adobe Indesign, it automates the rhythmic alternation between long and short lines. Simply set your desired width for even and odd lines, and let the optical algorithm calculate the best breaks to achieve a perfect, orderly typographic texture.

How to Install: Place the .jsx file directly into the Scripts Panel folder of Adobe InDesign.
Include the .png file as well if you want the mascot!

(The typical path is: Applications > Adobe InDesign > Scripts > Community > Scripts Panel) ### How to Use:
Select a text frame with justified text and have fun with Swissy!
